
# Frontend (React)

This directory contains the React frontend for the Developer Portfolio Website. It provides a modern, responsive UI for showcasing projects, skills, and music.

## Features
- Dynamic project display (fetched from backend API)
- Skills and About sections
- Music player integration
- Contact form and social links
- Fully tested with Jest

## Technologies
- React
- JavaScript
- CSS
- Jest (testing)

## Setup & Installation
1. Navigate to the frontend directory:
	```bash
	cd frontend
	```
2. Install dependencies:
	```bash
	npm install
	```
3. Start the development server:
	```bash
	npm start
	```

## Testing
Run all frontend tests with:
```bash
npm test
```

## Project Structure
- `src/` — Main source code
- `public/` — Static assets
